# MetricISO
MetricISO es un dashboard web que automatiza métricas de calidad de ISO 29110, 9001, 25010 y 9241. Analiza repositorios GitHub para obtener métricas de commits, complejidad ciclomática, LOC y 13 fórmulas ISO, y almacena historial por proyecto con una arquitectura basada en Flask, React y MySQL.
